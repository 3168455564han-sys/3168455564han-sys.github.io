关于这个站点

这里不是传统博客首页，而是一套更接近知识库的静态笔记网站。

我会把网络安全学习、面试整理、阶段性复盘和零散想法都收进这里，用 Markdown 持续维护。

使用方式

  • 笔记正文放在 source/_posts/
  • 独立页面放在 source/
  • 构建输出目录是 public/
  • 你可以把 public/ 的内容直接上传到 GitHub Pages

目标

把这个站点长期当成公开的个人笔记库，而不是只放几篇孤立文章。
